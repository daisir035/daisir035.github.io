  1. 从 github 克隆sllidar_ros2包

    在克隆之前，请确保您仍在 ros2_ws/src 目录中：

    git clone https://github.com/Slamtec/sllidar_ros2.git

  2. 构建sllidar_ros2包

    从工作区的根目录 （ros2_ws），现在可以使用以下命令生成sllidar_ros2包：

    cd ~/ros2_ws/
source /opt/ros/<rosdistro>/setup.bash
colcon build --symlink-install

    如果您找到类似“colcon：command not found”的输出，则需要单独的安装 colcon 构建工具。

  3. 包环境设置

    source ./install/setup.bash

    注： 添加永久工作区环境变量。 如果每次启动新 shell 时都会自动将 ROS2 环境变量添加到您的 bash 会话中，这很方便：

echo "source <your_own_ros2_ws>/install/setup.bash" >> ~/.bashrc
source ~/.bashrc

  1. 为 rplidar 创建 udev 规则

    sllidar_ros2运行需要串行设备的读写权限。 您可以使用以下命令手动修改它：

    sudo chmod 777 /dev/ttyUSB0

    但更好的方法是usb自动鉴权
